-src 
- components : Chứa dump components , các components dùng để sử dụng bên trong các component khác , thông thường chỉ nhận props và các render UI , ít sử lý logic , Ex : Button , Card, div
- pages : Chứa các Component cấu thành 1 page , có các xử lý logic bên trong những component này , như gọi API 
- hook : Chứa các custom hooks , useRequest, useABC 
- services: nơi setup API
- style : nơi đặt các file css/scss
 + Sử dụng scss trong react : npm i -D sass 
 + Utils : Chứa các common function để tái sử dụng trong project
+ Layouts : Chứ các component layout
+ auth: Chứa các component và các hàm xử lý liên quan đến authentication

 * Có redux 
 - contants : Chứa contants của project và type của redux
 - action : Chứa action của redux 
 - reducers : Chứa reducers của redux
